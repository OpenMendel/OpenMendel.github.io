OpenMendel

An Open Source Program for Statistical Genetics Analysis

OpenMendel Packages

OpenMendel is an Open Source project implemented in the Julia programming language that comprises a set of packages for statistical analysis to solve a variety of genetic problems.

Packages:

  • AIM Selection selects the SNPs that are most informative at predicting ancestry for your data â€” the best Ancestry Informative Markers (AIMs).
  • Estimate Frequencies calculates a likelihood-based estimate of allele frequencies.
  • Gamete Competition analysis.
  • Gene Dropping performs gene dropping with several useful options for the output.
  • Genetic Counseling performs risk calculates for genetic counseling problems.
  • GWAS performs standard Genome-wide Association Studies.
  • Kinship computes kinship and other identity coefficients.
  • Location Scores maps a trait via the method of Location Scores, i.e., multipoint linkage analysis.
  • Two Point Linkage performs two-point linkage analysis.
  • Regression Tests - Collection of test cases for OpenMendel framework
  • Variance Component Models - Utilities for fitting and testing variance component models

OpenMendel also includes the following utility packages

Utilities:

  • Search is a program for function optimization. Search permits bounds and linear constraints to be imposed on parameters and, in statistical applications, computes asymptotic standard errors and correlations of parameter estimates.
  • MendelBase contains the base functions of OpenMendel
  • SnpArrays provides utilities for handling compressed storage of biallelic SNP data.

Installation

OpenMendel runs on Julia v0.4. The three OpenMendel Utility packages - SnpArrays, Search, and MendelBase must be installed before any Mendel analysis packages will run.

Within Julia, use the package manager to install the OpenMendel Packages:

    Pkg.clone("git@github.com:OpenMendel/MendelPackageName.jl.git")

Data Files

To run the analysis packages you will need to prepare a Control file and have your data files available. The Control file holds the names of your data files and any optional parameters and keywords needed for the analysis. Descriptions of the specific options available within the OpenMendel packages are in the documentation for each package.

There are example data files in the "docs" subfolder of the MendelPackageName package, for example, ~/.julia/v0.4/MendelPackageName/docs.

Running the Analysis

To run an analysis package, first launch Julia. Then load the package with the command:
    julia> using MendelPackageName
Next, if necessary, change to the directory containing your files, for example,
    julia> cd("~/path/to/data/files/")
Finally, to run the analysis using the parameters in the control file Control_file.txt use the command:
    julia> PackageName("Control_file.txt")
Note: The package is called MendelPackageName but the analysis function is called simply PackageName.

Citation

If you use this analysis package in your research, please cite the following reference in the resulting publications:

    Lange K, Papp JC, Sinsheimer JS, Sripracha R, Zhou H, Sobel EM (2013) Mendel: The Swiss army knife of genetic analysis programs. Bioinformatics 29:1568-1570.
