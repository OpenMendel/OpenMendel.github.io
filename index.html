OpenMendel

An Open Source Program for Statistical Genetics Analysis

OpenMendel Packages

OpenMendel is an open source project implemented in the [Julia](http://julialang.org/) programming language that comprises a set of packages for statistical analysis to solve a variety of genetic problems. OpenMendel is the open source version of the [Mendel]((http://www.genetics.ucla.edu/software/mendel)) [Mendel](http://www.genetics.ucla.edu/software/mendel), a comprehensive package for exact statistical genetic analysis of qualitative and quantitative traits. Mendel offers 24 options for statistical genetic analysis. OpenMendel currently provides the analysis packages listed below, with additional options under development. We invite the genetics community to contribute to the OpenMendel Project.

Packages:

  • AIM Selection selects the SNPs that are most informative at predicting ancestry for your data â€” the best Ancestry Informative Markers (AIMs).
  • Estimate Frequencies calculates a likelihood-based estimate of allele frequencies.
  • Gamete Competition analysis implements a gamete competition analysis, which is a generalization of the TDT analysis.
  • Gene Dropping performs gene dropping with several useful options for the output.
  • Genetic Counseling performs risk calculates for genetic counseling problems.
  • GWAS performs standard Genome-wide Association Studies.
  • Kinship Kinship computes kinship and other identity coefficients.
  • Location Scores maps a trait via the method of Location Scores, i.e., multipoint linkage analysis.
  • Two Point Linkage performs two-point linkage analysis.

Utilities

OpenMendel also includes the following utility packages:

  • Search is a program for function optimization. Search permits bounds and linear constraints to be imposed on parameters and, in statistical applications, computes asymptotic standard errors and correlations of parameter estimates.
  • MendelBase contains the base functions of OpenMendel
  • SnpArrays provides utilities for handling compressed storage of biallelic SNP data.
  • Variance Components Models provides utilities for fitting and testing variance components models, also known as linear mixed models.

Installation

Note: Three OpenMendel packages - SnpArrays, Search, and MendelBase must be installed before any Mendel analysis packages will run.

Within Julia, use the package manager to install MendelPackageName:

Pkg.clone("git@github.com:OpenMendel/MendelPackageName.jl.git")

This package supports Julia v0.4.

Data Files

To run an analysis package you will need to prepare a Control file and have your data files available. The Control file holds the names of your data files and any optional parameters for the analysis. Details on the general format and contents of the Control and data files can be found on the MendelBase documentation page. Descriptions of the specific options available within the any analysis package are in its documentation page.

There are example data files in the "docs" subfolder of each Mendel package, for example, ~/.julia/v0.4/MendelGWAS/docs.

Running the Analysis

To run this analysis package, first launch Julia. Then load the package with the command:

 julia> using MendelGeneDropping

Next, if necessary, change to the directory containing your files, for example,

 julia> cd("~/path/to/data/files/")

Finally, to run the analysis using the parameters in the control file Control_file.txt use the command:

 julia> GeneDropping("Control_file.txt")

Note: the package is called MendelGeneDropping but the analysis function is called simply GeneDropping.
